# NEO TOKYO CITY

![main](image/main.jpg)

## 概要
仮想電脳都市「NEO TOKYO CITY」をテーマにした体験型Webインターフェース作品。

ユーザーはID認証を通じて都市へアクセスし、
マップ移動・管区閲覧・決済システムなどを通じて、
サイバー空間に入り込むような体験を提供します。

## 特徴

- ID選択によるログイン体験
- マップ型ナビゲーションUI
- 管区ごとのビジュアル・テーマ変化
- ストレス値によるUI変化・演出変化
- AIキャラクター「寝音亜衣」によるガイド演出
- 決済システム（購入UI + スキャン演出）
- スラム侵入時のグリッチ演出

---

## こだわり

「Webサイト」ではなく、
**“体験できるUI”を作ること**を目的に設計しました。

特に以下を重視しています。

- ユーザーが“操作する楽しさ”を感じるUI
- 状態（ストレス値）によって変化する演出
- AIキャラクターによる没入感の強化
- サイバー世界観の統一されたデザイン

視覚・動き・状態変化を組み合わせることで、
単なる閲覧ではなく「体験」として成立させています。
